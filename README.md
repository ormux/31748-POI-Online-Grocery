<h3 align="center">Instructions</h3>
Before deployment make sure to create a `config.php` with the properties of your database.

```
<?php
$DB_URL  = <database url>
$DB_NAME = <database name>
$DB_USER = <database user>
$DB_PASS = <database pass>
?>
```
